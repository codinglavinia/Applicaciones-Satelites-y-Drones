# Desarollo de Applicaciones para Satélites y Drones
# 🛰️🌍 🚁

Repositorio dedicado a prácticas, ejercicios y proyectos de Teledetección, Sistemas de Información Geográfica (SIG/GIS) y Observación de la Tierra utilizando Python y Jupyter Notebook.

## 📚 Contenidos :

- 🛰️ Procesamiento de imágenes satelitales
- 🌱 Cálculo de índices de vegetación (NDVI)
- 🗺️ Análisis de datos geoespaciales
- 📡 Observación de la Tierra
- 🚁 Aplicaciones con datos de drones (UAV)
- 📈 Visualización y análisis de datos raster

## 🛠️ Tecnologías y Herramientas :

- 🐍 Python
- 📓 Jupyter Notebook
- 🔢 NumPy
- 📊 Matplotlib
- 🗺️ Rasterio
- 🌍 GDAL
- 🧭 QGIS
